Game Info

Try to get through the tornado by moving around and pressing on keyboard buttons.


Controls: [Spoiler ahead!!!!]

Down arrow = Duck.

Up arrow = Jump.

Left arrow = Move left.

Right arrow = Move right.

End key = Changes face.

Move to the right end of the barrels + shift = Sword slash.

Move to the left end of the barrels + ctrl = Lance stab.

Space= Die.
